|                          |                                                                                             |
|--------------------------|---------------------------------------------------------------------------------------------|
| Условный оператор        | `if-elseif-else-end`                                                                        |
| Простой цикл `for`       | `for i in 1:10`<br>`    println(i)`<br>`end`                                                |
| Не вложенные циклы for   | `for i in 1:10, j = 1:5`<br>`    println(i*j)`<br>`end`                                     |
| Перечисление             | `for (idx, val) in enumerate(arr)`<br>`    println("the $idx-th element is $val")`<br>`end` |
| Цикл `while`             | `while bool_expr`<br>`    # do stuff`<br>`end`                                              |
| Выйти из цикла           | `break`                                                                                     |
| Завершить итерацию цикла | `continue`                                                                                  |
